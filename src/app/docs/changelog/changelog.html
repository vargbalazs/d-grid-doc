Changelog history

v1.0.3 (27.10.2025)

  • adding filtering functionality with the filterable property (true/false, row, menu, excel)

v1.0.2-a (24.09.2025)

  • adding resizable property and feature
  • configuring column resize behaviour with minResizableWidth and maxResizableWidth
  • autosize column upon double click on the resize element
  • introducing constrained mode

v1.0.2 (18.09.2025)

  • introducing style tag for holding styles
  • laying down the foundation of using internal css variables
  • sticky header and footer
  • error template
  • icon service and using custom icons
  • defining columns with columns signal or with d-grid-column elements
  • column width features (width, autosize, minAutoColWidth)
  • defining column field and header
  • change column visibility with the hidden property
  • column sorting features:
    • single and multiple sort modes
    • swith to multiple sorting with the Ctrl key
    • introducing sortable property
    • many options like compareFn, sortValueFn, initialDir, sortPriority
  • column templates (header, cell, footer)
  • defining grid data (static or asynchronous)
  • loading template
  • no data template
  • sticky rows
  • animated rows
