DataGrid aka DGrid - the new grid library out there

General purpose

This grid library was born in order to have a performant grid library, which don't uses zone.js, uses OnPush change detection and uses signals internally and also externally for it's inputs (if it makes sense). The grid library was written in Angular for only Angular.

Dependencies

As for now, the library uses the following dependencies:

  • uuid: for generating unique ids for columns and rows

Internal structure

The grid uses the modern css grid layout with divs instead of classic table elements like <th>, <tr> and <td>. Only the main semantic elements (header, body, footer) are real Angular components in order to minimize change detection overhead and have better performance.
