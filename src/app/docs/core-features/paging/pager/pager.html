Pager

To enable the paging feature in the grid, set the pageable property to true. Setting this will show a pager component below the footer of the grid. With the pageSize property we can define, how many rows will be showed in the grid (default is 10). Alternatively we can define an array of pageSizes. This will show a dropdown in the pager and we can select the desired page size.

Page mode

With the pageMode setting we can define, how grouping works with paging together. If pageMode is set to 'rows' (the default), grouping will be applied to each page separately. If pageMode is set to 'groups', the grid data will be grouped first and then the top level groups will be the base of paging.
