Column configuration

Define columns

You can define the grid columns either with an input property, which is linked to an array, or using d-grid-column elements.

Define columns with input

To define columns with an input property you have to define a column array first, which has to be a signal array, f. e.:

The library exposes the DGridColumn interface, which has the following members:

  • field: required / defines the field to be shown from the data source
  • header: required / defines the header text
  • sortable: defines the sortable behaviour of the grid (for more see 'Sorting')
  • width: defines the width of the column
  • autosize: defines whether the column resizes based on it's content
  • hidden: defines whether the column should be not visible
