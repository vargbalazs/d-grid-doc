Filter like in Excel

If we set the filterable property to excel, a filter icon appears in each column header. Clicking on this icon, a window is showed, which provides Excel like functionality.

Filter operators

For each data type there is a set of filter operators, which can be used to filter the grid data.

Number filter operators

  • Is equal to
  • Is not equal to
  • Is greater than or equal to
  • Is greater than
  • Is less than or equal to
  • Is less than
  • Is null
  • Is not null

String filter operators

  • Is equal to
  • Is not equal to
  • Contains
  • Does not contain
  • Starts with
  • Ends with
  • Is null
  • Is not null
  • Is empty
  • Is not empty

Date filter operators

  • Is equal to
  • Is not equal to
  • Is after or equal to
  • Is after
  • Is before or equal to
  • Is before
  • Is null
  • Is not null

Boolean filter operators

  • Is True
  • Is False
  • Show All
