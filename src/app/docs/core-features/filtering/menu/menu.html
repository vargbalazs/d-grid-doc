Filter menu

If we set the filterable property to menu, a filter icon appears in each column header. Clicking on this icon, we can define filters for the given column. Also in this case editors are rendered according to the data type:

  • number: text editor with only numbers, minus sign and comma or dot (depending on the locale)
  • string: text editor
  • boolean: dropdown
  • date: default date editor

In the showed window we can choose a filter operator. Depending on the data type, there are different filter operators available. We can also combine different operators by clicking on the plus button. In this case the operators are grouped in appearance order. F. e. if we set a condition A with an other condition B combined with AND and set a third condition C with OR, then it will be evaulated like this: (A AND B) OR C.

Filter operators

For each data type there is a set of filter operators, which can be used to filter the grid data.

Number filter operators

  • Is equal to
  • Is not equal to
  • Is greater than or equal to
  • Is greater than
  • Is less than or equal to
  • Is less than
  • Is null
  • Is not null

String filter operators

  • Is equal to
  • Is not equal to
  • Contains
  • Does not contain
  • Starts with
  • Ends with
  • Is null
  • Is not null
  • Is empty
  • Is not empty

Date filter operators

  • Is equal to
  • Is not equal to
  • Is after or equal to
  • Is after
  • Is before or equal to
  • Is before
  • Is null
  • Is not null

Boolean filter operators

  • Is True
  • Is False
  • Show All
