Filter row

If we set the filterable property to true or row, a filter row is showing below the header. For each data type a default editor will be rendered:

  • number: text editor with only numbers, minus sign and comma or dot (depending on the locale)
  • string: text editor
  • boolean: dropdown
  • date: default date editor

Clicking on the icon left to the rendered editor we can choose a filter operator. Depending on the data type, there are different filter operators available.

Filter operators

For each data type there is a set of filter operators, which can be used to filter the grid data.

Number filter operators

  • Is equal to
  • Is not equal to
  • Is greater than or equal to
  • Is greater than
  • Is less than or equal to
  • Is less than
  • Is null
  • Is not null

String filter operators

  • Is equal to
  • Is not equal to
  • Contains
  • Does not contain
  • Starts with
  • Ends with
  • Is null
  • Is not null
  • Is empty
  • Is not empty

Date filter operators

  • Is equal to
  • Is not equal to
  • Is after or equal to
  • Is after
  • Is before or equal to
  • Is before
  • Is null
  • Is not null

Boolean filter operators

  • Is True
  • Is False
  • Show All
