Column filtering

In order to filter the grid data, we can use the filterable property of the grid. This can have several values:

  • true/false: in case of true the default filter mechanism is applied, showing a filter row
  • row: same as true, a filter row is showed
  • menu: a filter icon is showed in the header for each column; clicking on it we can filter the column
  • excel: this setting provides excel-like filtering features
