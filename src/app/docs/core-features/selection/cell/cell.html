Cell selection

Default settings

As per default selection is enabled. Which means, you can select any cell with the mouse (hold and drag) or with Shift+Arrow keys.

If you want to change this behavior, just assign a DGridSelectionSettings object to the selectable property of the grid. Within this object you can fine tune the selection with this properties:

  • withKeyboard: takes a boolean value, enables/disables selection with the keyboard
  • withMouse: takes a boolean value, enables/disables selection with the mouse

Please note, if navigation is not allowed (navigable is set to false) selection with the keyboard won't work, regardless the selection settings.
