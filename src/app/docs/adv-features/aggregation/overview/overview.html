Aggregation overview

You can show aggregates either in group header or group footer templates.

Currently the following aggregates are supported:

  • sum: calculates the total of all numeric values in the group
  • count: returns the number of items in the group
  • min: finds the smallest value in the group
  • max: finds the largest value in the group
